# MadLibs Game

A fun word game where players fill in blanks to create hilarious stories!

## Features

- Two interfaces: Command Line (CLI) and Graphical (GUI)
- Multiple story templates
- Colorful and user-friendly interface
- Easy to play and extend

## How to Run

1. Make sure you have Python 3.x installed
2. Run the main program:
python main.py

text
3. Choose your preferred interface (CLI or GUI)

## How to Play

1. Select a mode (CLI or GUI)
2. Fill in the requested words (adjectives, nouns, verbs, etc.)
3. See your custom story generated!
4. Play again with a new story

## Adding New Templates

You can extend the game by adding new story templates to the `default_templates` list in `madlibs/core.py`.

## Requirements

- Python 3.x
- tkinter (usually included with Python)

For Linux users, you might need to install tkinter:
- Ubuntu/Debian: `sudo apt-get install python3-tk`
- Fedora: `sudo dnf install python3-tkinter`
